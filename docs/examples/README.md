# Workflow Examples

## Purpose
This directory contains fictional, practical examples of workflow audits, automation blueprints, and AI agent designs.

## Files
- `workflows/sample-lead-follow-up-audit.md`: Customer-facing, draft-first example.
- `workflows/sample-client-onboarding-blueprint.md`: Handoffs and document collection example.
- `workflows/sample-document-collection-agent.md`: Safe agent design for sensitive documents.
- `workflows/sample-invoice-reminder-workflow.md`: Finance-adjacent, human-in-the-loop example.
- `workflows/sample-weekly-reporting-automation.md`: Internal reporting automation.

## How to use
Use these examples as references when filling out templates from the `knowledge-files` or `prompts` directories. They demonstrate the expected level of detail, the tone, and the strict adherence to safety and human-in-the-loop principles.

## Safety note
All companies and scenarios described here are fictional. They serve to illustrate concepts. Ensure any real-world implementation includes appropriate legal, financial, or compliance review.
