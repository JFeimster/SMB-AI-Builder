# Safety Language

Use this language across connector/tooling docs and pages:

> This GPT supports workflow planning and pilot design. It is not legal, financial, tax, HR, lending, accounting, medical, underwriting, or compliance advice. Sensitive decisions should stay under qualified human review.

## Avoid

- Guaranteed results
- 100% accuracy
- Fully autonomous decisions
- Final eligibility decisions
- Final accounting conclusions
- Final legal/tax/HR/compliance recommendations
- Live integration claims unless verified

## Use Instead

- Draft-first
- Human-reviewed
- Pilot plan
- Review checkpoint
- Risk classification
- Minimum connector stack
- Workflow map
- Safe sample-data test
