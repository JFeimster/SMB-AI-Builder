# Merge Notes

This branch is intentionally smaller and cleaner than the combined stale PR pile.

## Merge Intent

Ship a stable connector/tooling foundation now, then build the deeper connector-specific docs, UI refactor, OpenAPI/action layer, and tests in focused follow-up PRs.

## Why Not Merge The Old PRs?

The old branches were stale, conflicted, or depended on each other. Several included broken links, sitemap formatting issues, or known bugs. Rebuilding from current `main` avoids dragging those problems forward.

## What This Branch Adds

- Connector docs indexes
- Connector risk rules
- Workflow connector map index
- Connector stack index
- Prompt pack index
- Selected playbooks, maps, stacks, and prompts
- Public connector and skills directory pages
- Sitemap entries
- Selected per-tool skills
- Deferral docs for UI, tests, and OpenAPI/action work

## Human Review

Review public page paths, sitemap XML, and safety language before merge.
