# Prompt Documentation

## Purpose
This directory contains reusable prompt libraries for the SMB AI Workflow & Agent Builder. These prompts guide the analysis, design, and testing of workflows and AI agents.

## Files
- `audit-prompts.md`: Prompts for analyzing current processes.
- `readiness-prompts.md`: Prompts for evaluating automation readiness.
- `automation-blueprint-prompts.md`: Prompts for designing automated workflows.
- `agent-blueprint-prompts.md`: Prompts for defining AI agent behaviors.
- `testing-prompts.md`: Prompts for QA and adversarial testing.
- `demo-prompts.md`: Prompts for generating sample content and sales materials.
- `landing-page-package.md`: Instructions for generating landing pages.
- `agent-launch-site-builder.md`: Instructions for agent launch sites.

## How to use
Copy and paste the specific prompt blocks into your GPT or LLM interface. Modify the context within brackets `[]` to fit your specific scenario. Use the "Follow-up prompts" to dig deeper into the analysis.

## Safety note
These prompts are tools for planning. The outputs generated by AI should always be reviewed by a qualified human, especially when dealing with sensitive, financial, or compliance-related workflows. The AI provides drafts, humans make decisions.
