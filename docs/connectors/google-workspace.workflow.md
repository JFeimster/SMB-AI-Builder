# Google Workspace Connector Playbook

## Connector

Google Workspace: Drive, Gmail, Calendar, and Contacts.

## Best For

- Document collection
- Email follow-up
- Meeting-to-task workflows
- Client onboarding
- Internal weekly reporting
- Drafting checklists and summaries

## Do Not Use For

- Final legal, tax, HR, lending, underwriting, accounting, medical, or compliance decisions
- Automatic customer-facing sends without review
- Processing private files without permission

## Required Inputs

- Workflow goal
- Data source
- Reviewer/owner
- Output destination
- Sensitive data risks

## Common SMB Workflows

- Missing document follow-up
- Meeting recap to tasks
- Weekly operations summary
- Client onboarding checklist
- Sales call follow-up draft
- Proposal request routing

## Automation Opportunities

- Draft emails
- Summarize notes
- Generate checklists
- Route files to review queues
- Create task drafts

## What Should Stay Human

- Final sends
- Sensitive decisions
- Customer commitments
- Financial, legal, HR, or compliance conclusions

## Privacy / Security Risks

Drive and Gmail can contain private customer, financial, legal, or HR data. Start with sample data, restrict scope, and keep outputs draft-first.

## Risk Level

Medium by default. High when customer, money, legal, HR, or regulated data is involved.

## Starter Prompt

```text
Map this Google Workspace workflow across Drive, Gmail, Calendar, and Contacts. Identify the minimum stack, draft-safe outputs, human review checkpoints, and the first safe pilot.
```

## Example Output

- Workflow map
- Connector stack
- Draft outputs
- Human review points
- Risk notes
- Pilot checklist
