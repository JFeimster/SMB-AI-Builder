# Recommended Knowledge Files Package

## Overview
This package contains 22 required files for the SMB AI Workflow & Agent Builder project. It provides frameworks, templates, JSON schemas, UI mockups, and practical guides for auditing SMB workflows and building safe automation pilots.

## Suggested Use
These files should be uploaded as Knowledge Files to the Custom GPT to train it on the specific methodologies, tone, and safety rules required for SMB automation consulting.

## Safety Notes
All workflows that touch money, legal, HR, medical, compliance, customer communication, or private data MUST require human review. The GPT must not provide professional compliance advice.
