# Assets Directory

This directory stores public-facing media, images, and brand assets for the project.

## Subdirectories
- `icons/`: Store SVG icons used in the UI.
- `logos/`: Store brand logos and partner logos (must be real, no fake logos).
- `banners/`: Store wide hero images or social sharing banners.
- `screenshots/`: Store actual product screenshots (must be real, no fake screenshots).
- `diagrams/`: Store workflow maps, architectural diagrams, and flowchart graphics.

## Naming Conventions
Use lowercase letters, numbers, and hyphens only.
Example: `workflow-audit-dashboard-v1.png`

## Recommended Formats
- **Icons:** SVG
- **Logos:** SVG or transparent PNG
- **Screenshots:** PNG or WebP
- **Diagrams:** SVG or high-res PNG

## Image Optimization
Please compress images before committing to reduce repository bloat. Tools like TinyPNG or ImageOptim are recommended for raster formats.

## Accessibility
Ensure all decorative images have `alt=""` and meaningful images have descriptive `alt` text in the HTML referencing them.

## Important: No Fake Proof Rule
Do not add fake logos, fake screenshots, fake partner marks, fake certification badges, fake ratings, or fake customer proof.
Note: A favicon has not been added yet and should be placed in `public/` when available.
