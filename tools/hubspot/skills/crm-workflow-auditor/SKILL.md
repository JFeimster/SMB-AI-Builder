# Skill: CRM Workflow Auditor

## Purpose
Audits HubSpot CRM setups and proposes workflow optimizations.

## Use When
When a user wants to improve their HubSpot usage, clean up data, or streamline sales processes.

## Inputs Needed
- Current HubSpot usage and tier
- Sales or marketing goals
- Identified bottlenecks or data issues

## Outputs
- An audit report of current practices
- Recommendations for custom properties or pipeline stages
- Data cleanup strategies

## Procedure
1. Review the user's current CRM setup and processes.
2. Identify areas where standard HubSpot features can replace manual work.
3. Propose updates to Lifecycle Stages, Lead Status, or Deal Pipelines.
4. Suggest data hygiene practices (e.g., requiring certain fields).
5. Verify against safety and advice rules.

## Safety / Human Review
- No financial, legal, or HR advice.
- Do not claim to automatically modify their HubSpot instance.
- Avoid suggesting enterprise features if the user is on a starter tier.

## Quality Checklist
- [ ] Recommendations fit the user's tier?
- [ ] Pipeline stages are logical?
- [ ] Data hygiene is addressed?

## Example Prompt
```
Our sales team is using HubSpot, but our deal pipeline is a mess and we have a lot of stale contacts. What are some best practices for organizing our pipeline stages and cleaning up our database?
```
