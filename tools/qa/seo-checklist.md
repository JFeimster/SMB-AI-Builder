# SEO Checklist

Ensure pages are optimized for search engines according to project rules.

## Core Checks
- [ ] **Uniqueness**: Confirm Title and Meta Description do not duplicate other pages on the site.
- [ ] **H2 Structure**: H2s are descriptive and outline the content logically.
- [ ] **JSON-LD Schema**: Valid JSON-LD schema (e.g., `Article` and `FAQPage`) is present where appropriate (especially on comparison and workflow audit pages).
- [ ] **Internal Linking**: Page links to other relevant content within the site to establish architecture.
- [ ] **Robots/Sitemap**:
  - Page is included in `sitemap.xml` if it should be indexed.
  - Page is NOT blocked by `robots.txt` unless intentionally excluded.
- [ ] **Page Rules**:
  - SEO pages include consistent header/nav/footer.
  - SEO pages contain 4-6 FAQs.
  - SEO pages include a CTA band near the end.
