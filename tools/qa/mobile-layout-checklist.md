# Mobile Layout Checklist

Ensure the site is fully usable and visually correct on narrow viewports.

## Core Checks
- [ ] **Viewport meta tag**: Ensure `<meta name="viewport" content="width=device-width, initial-scale=1">` is present.
- [ ] **No horizontal scrolling**: Content does not break out of the viewport width; users do not need to scroll horizontally.
- [ ] **Touch targets**: Buttons, links, and navigation items are large enough to be easily tapped (minimum 44x44 CSS pixels).
- [ ] **Readability**: Text scales correctly and remains readable without zooming.
- [ ] **Navigation**: Mobile menus (if applicable) open, close, and function correctly.
- [ ] **Spacing**: Padding and margins collapse gracefully; content does not look cramped.
- [ ] **Tables/Grids**: Comparison tables or data grids resize, stack, or allow localized horizontal scrolling without breaking the main page layout.
