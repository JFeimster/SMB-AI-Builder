# Embed Checklist

Use this checklist when QAing iframe embeds meant for partners or external deployment.

## Core Checks
- [ ] **Max Width**: Embed does not exceed 940px width.
- [ ] **Max Height**: Embed does not exceed 1200px height.
- [ ] **Compact Mode**: If designed for narrow columns, the compact CSS rules apply correctly.
- [ ] **Partner Attribution**: Any required "Powered by SMB AI Builder" or partner-specific branding is visible and legible.
- [ ] **UTM/Ref Params**: Links within the embed carry correct UTM or referral parameters so traffic can be tracked back to the embed source.
- [ ] **Disclaimer**: The required sensitive workflow disclaimer is visible if the embed touches on legal, financial, HR, or other regulated topics.
