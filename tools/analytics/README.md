# Analytics Layer

This directory (`tools/analytics/`) contains the privacy-aware analytics planning layer for the SMB AI Builder site and partner embeds.

The primary purpose of this layer is to outline what metrics we plan to track (CTAs, embeddings, etc.), how UTM parameters should be structured for partner tracking, and strictly enforce privacy rules so that no sensitive workflow details or personal data are ever collected.

**Important Notes:**
- No actual analytics scripts or tracking tags should be added without explicit instruction.
- No real tracking IDs should be included in these files.
- No analytics data is invented.
- No sensitive workflow contents are collected.
