# CTA Tracking Plan

This document outlines the tracking strategy for Call-To-Action (CTA) elements across the site and partner embeds.

## Tracked CTAs

- **GPT launch CTA**: Tracks clicks leading to the GPT launch or interaction.
- **Notion resource CTA**: Tracks clicks leading to Notion resources (e.g., Notion workspace).
- **GitHub resource CTA**: Tracks clicks leading to the GitHub repository or related GitHub resources.
- **Partner embed CTA**: Tracks clicks on CTAs located within partner embeds.
