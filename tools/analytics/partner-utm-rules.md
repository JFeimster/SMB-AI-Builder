# Partner UTM Rules

This document defines the standardized UTM parameter rules for tracking partner traffic and embeds.

## Required Parameters

- `utm_source`: Always set to `partner`
- `utm_medium`: Always set to `embed`
- `utm_campaign`: Always set to `smb_ai_builder`
- `utm_content`: Set to `[partner-id]` (the unique identifier for the partner)
- `ref`: Set to `[partner-id]` (the unique identifier for the partner)
