# Analytics Event Map

This file defines the optional analytics events that may be tracked across the SMB AI Builder site and partner embeds.

## Defined Events

- `primary_cta_click`: Triggered when the main CTA is clicked.
- `secondary_resource_click`: Triggered when a secondary resource link is clicked.
- `github_resource_click`: Triggered when a GitHub resource link is clicked.
- `faq_opened`: Triggered when an FAQ item is expanded or opened.
- `nav_click`: Triggered when a navigation link is clicked.
- `readiness_picker_started`: Triggered when the readiness picker interaction starts.
- `readiness_picker_completed`: Triggered when the readiness picker interaction is completed.
- `partner_embed_loaded`: Triggered when a partner embed successfully loads.
- `partner_embed_cta_click`: Triggered when a CTA within a partner embed is clicked.
- `partner_resource_click`: Triggered when a partner resource link is clicked.
