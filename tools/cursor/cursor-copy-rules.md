# Cursor Copy Rules

When writing or modifying copy, strictly adhere to the following rules:

1. **Tone:** Direct, practical, anti-hype, SMB-friendly.
2. **No Fluff:** Avoid fluffy deployment theory. Focus on actionable insights and practical checklists.
3. **Accuracy:** Do not invent facts, metrics, testimonials, or claims.
4. **Safety Disclaimers:** For sensitive workflows (finance, HR, legal, etc.), explicitly state that the tool provides planning support, not professional advice.
5. **CTAs:** Use clear, action-oriented primary CTAs (e.g., "Audit My First Workflow", "Score My Automation Readiness").
