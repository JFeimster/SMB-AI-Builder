# Claude QA Checklist

Before finalizing any document, code, or review, ensure the following checks have been completed:

- [ ] **No Unsupported Claims:** Confirm no fake testimonials, ratings, logos, revenue claims, compliance claims, case studies, or guarantees have been invented.
- [ ] **Safety Disclaimers:** Confirm that the explicit disclaimer is included for any sensitive workflows or planning documents.
- [ ] **No Professional Advice:** Confirm that GPT output is not framed as legal, financial, tax, HR, lending, accounting, or compliance advice.
- [ ] **Tone and Voice:** Confirm the copy is direct, no-BS, operator-friendly, and avoids vague AI hype words (e.g., "revolutionize", "unlock").
- [ ] **Link Verification:** Confirm all internal links use the correct relative paths and external canonical/OG URLs point to `https://smb-ai-builder.vercel.app`.
- [ ] **CTA Accuracy:** Confirm CTA links use the primary URL and core CTA language.
- [ ] **SEO Compliance:** Confirm unique titles, meta descriptions, one H1, descriptive H2s, valid JSON-LD schema, and 4-6 FAQs for SEO pages.
- [ ] **Code Integrity:** Confirm no secrets or private tokens are present in the code or documentation.
- [ ] **Design System:** For code/HTML, confirm it adheres to semantic HTML and the neobrutalist design system.
