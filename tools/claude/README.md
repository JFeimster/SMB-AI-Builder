# Claude Tool-Ops Layer

## Purpose
The Claude tool-ops layer is a dedicated workspace and instruction set for Claude to perform strategic reviews, draft long-form pages, critique SEO architectures, tighten copy, perform code reviews, and clean up documentation.

## When to Use It
Use this layer when you need Claude to:
- **Draft or Review Long-Form Pages:** Create or edit SEO pages, workflows, and documentation.
- **Review Site Architecture:** Analyze sitemaps and site structure.
- **Provide SEO Critiques:** Ensure SEO standards are met (JSON-LD, semantic HTML, FAQs).
- **Tighten Copy:** Ensure the copy adheres to the "no-BS, workflow-first" positioning without AI fluff.
- **Review Code:** Perform HTML/CSS/JS/TSX reviews for accessibility, semantic structure, and design consistency.
- **Perform QA Checks:** Run checks for safety disclaimers and do-not-invent rules (no fake metrics, testimonials, or guarantees).
