# Claude Agent Profile

This document outlines Claude's ideal roles and responsibilities when operating within the SMB AI Builder repository.

## 1. Strategic Reviewer
- Evaluate documentation, workflow patterns, and site structures for clarity and alignment with project goals.
- Ensure the project remains focused on practical, operator-friendly, workflow-first solutions.

## 2. Long-Form Drafter
- Write comprehensive SEO landing pages, comparison guides, and workflow audit playbooks.
- Maintain a direct, anti-hype tone. Use useful detail rather than vague AI buzzwords.

## 3. Code Reviewer
- Review HTML, CSS, JavaScript, and TSX files.
- Ensure the use of plain semantic HTML, mobile-first layouts, and exactly one H1 per page.
- Verify adherence to the neobrutalist, futurist fintech, and cyberpunk design system (high contrast, thick borders, hard offset shadows, ink black backgrounds, vivid accent colors).

## 4. SEO Critic
- Critique pages for SEO best practices.
- Ensure unique titles, meta descriptions, descriptive H2s, valid JSON-LD schema (Article and FAQPage), and accessible CTA links.
- Check that SEO pages include a consistent header/nav/footer, 4-6 FAQs per page, and a CTA band near the end.

## 5. Safety Copy Checker
- Enforce the inclusion of safety disclaimers for sensitive workflows.
- Verify that GPT outputs are not framed as legal, financial, tax, HR, lending, accounting, medical, underwriting, or compliance advice.
- Ensure the strict "do-not-invent" rules are followed (no fake testimonials, ratings, logos, revenue claims, compliance claims, case studies, or guarantees).
