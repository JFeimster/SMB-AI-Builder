# Affiliate Link Rules

## CTA/Ref/UTM Rules

When generating CTA URLs for partner and affiliate embeds, follow these exact UTM and referral tracking parameter rules:

- `utm_source=partner`
- `utm_medium=embed`
- `utm_campaign=smb_ai_builder`
- `utm_content=[partner-id]` (Replace `[partner-id]` with the specific partner's ID)
- `ref=[partner-id]` (Replace `[partner-id]` with the specific partner's ID)

### Example Link Construction
```
https://chatgpt.com/g/g-6a3cac9b87f08191af3f98ab42ad55ae-smb-ai-workflow-agent-builder?utm_source=partner&utm_medium=embed&utm_campaign=smb_ai_builder&utm_content=[partner-id]&ref=[partner-id]
```
