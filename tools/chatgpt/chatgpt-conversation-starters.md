# Conversation Starters

Use these quick conversation starters when interacting with ChatGPT to jumpstart tasks while attaching the relevant context files from this directory.

## Site Building & Design
> "I need to build a new static HTML tool page for the SMB AI Builder. Please read `chatgpt-project-context.md` and `chatgpt-page-generation-rules.md`. The tool is a 'Readiness Score Calculator'. Please output a wireframe outline first."

> "Act as a Landing Page Strategist. Review my current `index.html` (attached). I want to increase conversions to the Custom GPT. Suggest structural changes based on `chatgpt-copy-rules.md`."

## SEO & Content Generation
