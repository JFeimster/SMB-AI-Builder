# Skill: Content SEO Workflow Planner

## Purpose

Plan content and SEO workflows that connect research, briefs, pages, metadata, publishing, and refresh cycles.

## Use When

Use this skill when a workflow involves keyword research, blog planning, static pages, content briefs, internal links, schema, or content refreshes.

## Inputs Needed

- Topic or keyword
- Audience
- Existing URL/page
- Content goal
- Supporting assets
- Publishing workflow
- Review requirements

## Outputs

- Content workflow map
- Brief outline
- Internal link plan
- Metadata checklist
- Refresh plan
- Human review points

## Procedure

1. Define search intent and audience.
2. Identify existing assets.
3. Map content production steps.
4. Create metadata and link checklist.
5. Flag unsupported claims.
6. Recommend refresh cadence.

## Safety / Human Review

Do not invent rankings, traffic, testimonials, ratings, partner claims, or compliance claims. Content should be reviewed before publishing.

## Quality Checklist

- [ ] Search intent is clear.
- [ ] Claims are supportable.
- [ ] Metadata is included.
- [ ] Internal links are relevant.
- [ ] Publish review is included.

## Example Prompt

```text
Plan a content workflow for this topic. Include search intent, page outline, metadata, internal links, supporting assets, refresh checklist, and human review requirements.
```
