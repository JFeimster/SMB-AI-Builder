# Codex Style Guide

## Visual Direction
Neobrutalist + futurist fintech + cyberpunk.

The site should feel like a workflow war room, a fintech risk dashboard, a cyberpunk automation cockpit, and practical SMB operations meets neon AI infrastructure.

## Palette
- Ink Black: `#050505`
- Off White: `#F8F4E8`
- Acid Lime: `#C6FF00`
- Cyber Pink: `#FF2BD6`
- Electric Cyan: `#00E5FF`
- Signal Orange: `#FF7A00`
- Deep Violet: `#6D28D9`
- Hazard Yellow: `#FFE500`
- Brutalist Blue: `#1D4ED8`
- Concrete Gray: `#D1D5DB`

## Components
Use big bordered panels, thick outlines, hard offset shadows, high-contrast CTA buttons, scorecards, opportunity matrices, use-case cards, blueprint previews, terminal snippets, sticker badges, and neon outlines.

## Typography
Use huge hero headlines, heavy system sans fonts, monospace utility labels, short copy blocks, brutalist section labels, and big CTA text.

## Do Not Use
Generic AI orb visuals, corporate stock-photo layout, tiny CTAs, weak muted gray cards, or long brochure paragraphs.
