# SEO Schema Checker Skill

Rules:
- JSON-LD must be valid JSON.
- XML must be valid XML.
- Do not add fake ratings, reviews, aggregateRating, awards, user counts, or claims.
- Use placeholder strings only when a reusable template is requested.
- Add a note when production domain needs replacement.
