# Partner Embed Builder

## Purpose
Create and repair portable partner embed packages for SMB AI Builder.

## Use this skill when
- Working in `embeds/`.
- Creating full or compact iframe widgets.
- Reviewing partner attribution, UTM/ref tracking, embed CSS scope, or widget JS.

## Source-of-truth files to read first
- `tools/jules/JULES.md`
- `tools/jules/jules.json`
- `tools/jules/file-map.json`
- `tools/jules/copy-rules.md`
- `tools/jules/accessibility-checklist.md`

## Process
1. Confirm embed dimensions and portability.
2. Scope CSS to the embed wrapper.
3. Keep JS dependency-free and iframe-safe.
4. Preserve partner attribution and tracking without implying official partnerships.
5. Include compact safety/disclaimer language.

## Guardrails
- Do not duplicate the full landing page inside the embed.
- Do not invent partner relationships, affiliate claims, ratings, or integrations.
- Do not rely on root site CSS unless explicitly intended.
