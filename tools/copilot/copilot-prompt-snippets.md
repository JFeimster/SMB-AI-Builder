# Copilot Prompt Snippets

Use these snippets when interacting with GitHub Copilot to ensure responses align with the project context.

## Generate a new SEO Page structure
> "Generate the HTML skeleton for a new SEO workflow audit page targeting '[Keyword]'. Include the standard neobrutalist styling, 1 H1, the required sections (Why it's tempting, What can go wrong, Readiness factors), 4-6 FAQs, and JSON-LD schema. Ensure the CTA points to the official GPT URL and include the sensitive workflow disclaimer."

## Review HTML for Accessibility
> "Review this HTML block for accessibility. Ensure all images have alt attributes, form inputs have labels, contrast is adequate, and focus states are visible. Do not suggest adding external frameworks."

## Validate Tone and Content
> "Read the following copy. Remove any hype words, buzzwords (like 'unlock', 'revolutionize', 'seamless'), or fake claims. Rewrite it to be direct, operator-friendly, and practical, using checklists if applicable."

## Generate FAQ JSON-LD
> "Generate a valid JSON-LD FAQPage script block for the following questions and answers. Do not include any HTML, just the `<script type="application/ld+json">` block."
