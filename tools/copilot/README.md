# Copilot Tool-Ops Layer

## Purpose
This directory contains the GitHub Copilot tool-ops layer for the SMB AI Workflow & Agent Builder project.
It is designed to enforce consistent autocomplete behavior, PR reviews, issue templating, and overall repo contribution standards.

## Contents
- `copilot-review-checklist.md`: Specific items Copilot should check for during PR reviews or code audits.
- `copilot-prompt-snippets.md`: Useful prompt templates for interacting with Copilot in this codebase.
- `copilot-code-style.md`: Core rules for HTML/CSS/JS, the neobrutalist aesthetic, and Next.js standards.
- `copilot-seo-rules.md`: Strict SEO requirements, schema configurations, and URL structuring.
- `copilot-accessibility-rules.md`: Accessibility standards enforced across the site.
- `copilot-pr-rules.md`: Guidelines for opening and merging PRs.
- `copilot-changelog.md`: Template and rules for maintaining the project changelog.

## Integration
These files work in tandem with the `.github/copilot-instructions.md` file, which is automatically read by GitHub Copilot to set context for the repository.
