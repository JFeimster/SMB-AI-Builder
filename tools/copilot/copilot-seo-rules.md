# Copilot SEO Rules

## 1. URLs and Links
- All canonical URLs, Open Graph URLs (`og:url`), and Open Graph image URLs (`og:image`) must use **absolute paths** pointing to `https://smb-ai-builder.vercel.app`.
- Internal links should be relative based on directory depth (e.g., `../../index.html`).
- The primary CTA must ALWAYS be: `https://chatgpt.com/g/g-6a3cac9b87f08191af3f98ab42ad55ae-smb-ai-workflow-agent-builder`.

## 2. Meta Tags
- Every page must have a unique `<title>` and `<meta name="description">`.
- Descriptions should be concise, action-oriented, and free of hype buzzwords.

## 3. Page Structure
- **Exactly one `<h1>` per page.**
  - For Workflow Audit pages, the format MUST be: `"Audit Your [Workflow] Before Automating It With AI"`.
- Use descriptive `<h2>` and `<h3>` tags for section breaks.
- Include a consistent header, navigation, and footer on all SEO pages.
- Place a CTA band near the end of the page.

## 4. Structured Data
- Every SEO page must include valid JSON-LD schema blocks.
- **Required schemas:** `Article` and `FAQPage`.
- Schemas must reflect the actual content of the page accurately.

## 5. Content
- Include 4-6 FAQs per SEO page.
- Do not invent fake metrics, user testimonials, or integrations.
