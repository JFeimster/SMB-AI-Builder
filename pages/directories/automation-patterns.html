Automation Pattern Library

Automation Patterns You Can Reuse Without Overbuilding

Draft-Only Assistant

Use when: High risk, sensitive data.

Avoid when: Perfect accuracy isn't critical.

Checkpoint: Human approves every action.

Exception Router

Use when: Most tasks are standard, some are edge cases.

Avoid when: Every task is unique.

Checkpoint: Human handles the exceptions.

Straight-Through Processing

Use when: Zero risk, deterministic data.

Avoid when: LLMs are making decisions.

Checkpoint: Periodic audits.
