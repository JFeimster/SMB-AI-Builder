# extractWebsiteMetadata Action

This package contains a no-auth GPT Action endpoint for: Fetch public website metadata for SMB website and lead-flow audits.

## Files

```txt
app/api/site-metadata/route.ts
app/api/site-metadata/route.optional-endpoints.ts
app/api/site-metadata/site-metadata.ts
app/api/site-metadata/site-metadata-openapi.ts
app/api/site-metadata/site-metadata.openapi.json
app/api/site-metadata/site-metadata-request.example.json
```

## Setup

Vercel deploys files under `app/api/site-metadata/route.ts` as functions in a Next.js App Router project.

In GPT Builder:

```txt
Authentication: None
Schema: Paste the OpenAPI JSON.
```
