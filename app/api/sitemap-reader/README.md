# analyzeSitemap Action

This package contains a no-auth GPT Action endpoint for: Read a public sitemap to help understand a business website structure.

## Files

```txt
app/api/sitemap-reader/route.ts
app/api/sitemap-reader/route.optional-endpoints.ts
app/api/sitemap-reader/sitemap-reader.ts
app/api/sitemap-reader/sitemap-reader-openapi.ts
app/api/sitemap-reader/sitemap-reader.openapi.json
app/api/sitemap-reader/sitemap-reader-request.example.json
```

## Setup

Vercel deploys files under `app/api/sitemap-reader/route.ts` as functions in a Next.js App Router project.

In GPT Builder:

```txt
Authentication: None
Schema: Paste the OpenAPI JSON.
```
