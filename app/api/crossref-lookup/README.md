# lookupPublicationMetadata Action

This package contains a no-auth GPT Action endpoint for: Retrieve public citation metadata for articles, reports, DOI references, or research sources.

## Files

```txt
app/api/crossref-lookup/route.ts
app/api/crossref-lookup/route.optional-endpoints.ts
app/api/crossref-lookup/crossref-lookup.ts
app/api/crossref-lookup/crossref-lookup-openapi.ts
app/api/crossref-lookup/crossref-lookup.openapi.json
app/api/crossref-lookup/crossref-lookup-request.example.json
```

## Setup

Vercel deploys files under `app/api/crossref-lookup/route.ts` as functions in a Next.js App Router project.

In GPT Builder:

```txt
Authentication: None
Schema: Paste the OpenAPI JSON.
```
