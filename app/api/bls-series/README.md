# getLaborMarketSeries Action

This package contains a no-auth GPT Action endpoint for: Pull public labor/economic series for staffing, wage, or market-context discussions.

## Files

```txt
app/api/bls-series/route.ts
app/api/bls-series/route.optional-endpoints.ts
app/api/bls-series/bls-series.ts
app/api/bls-series/bls-series-openapi.ts
app/api/bls-series/bls-series.openapi.json
app/api/bls-series/bls-series-request.example.json
```

## Setup

Vercel deploys files under `app/api/bls-series/route.ts` as functions in a Next.js App Router project.

In GPT Builder:

```txt
Authentication: None
Schema: Paste the OpenAPI JSON.
```
