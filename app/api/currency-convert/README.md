# convertCurrencyEstimate Action

This package contains a no-auth GPT Action endpoint for: Estimate converted amounts across currencies for international SMB planning.

## Files

```txt
app/api/currency-convert/route.ts
app/api/currency-convert/route.optional-endpoints.ts
app/api/currency-convert/currency-convert.ts
app/api/currency-convert/currency-convert-openapi.ts
app/api/currency-convert/currency-convert.openapi.json
app/api/currency-convert/currency-convert-request.example.json
```

## Setup

Vercel deploys files under `app/api/currency-convert/route.ts` as functions in a Next.js App Router project.

In GPT Builder:

```txt
Authentication: None
Schema: Paste the OpenAPI JSON.
```
