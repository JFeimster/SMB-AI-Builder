# GitHub + Vercel Connector Prompts

## PR Queue Triage

```text
Review this PR queue and classify each PR as merge now, patch first, consolidate, or close as stale. Explain blockers and safest merge order.
```

## Static Site Readiness

```text
Audit this static site change for broken routes, missing sitemap entries, bad relative links, CSS/JS path problems, and deployment blockers.
```

## Repo Structure Cleanup

```text
Review this repo structure and recommend where public pages, internal tool docs, OpenAPI schemas, prompts, skills, and deployment files should live.
```

## Consolidation Plan

```text
Create a consolidation plan from stale branches. Start from current main, recreate only useful final state, fix broken links, and avoid stacking old branches.
```

## Vercel Readiness

```text
Create a Vercel readiness checklist for this repo. Include routes, sitemap, build artifacts, deployment settings, public assets, and rollback steps.
```
