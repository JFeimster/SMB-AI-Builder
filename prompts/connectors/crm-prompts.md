# CRM Connector Prompts

## Pipeline Audit

```text
Audit this CRM pipeline. Identify broken stages, missing fields, stale records, follow-up gaps, human review points, and a safe first automation pilot.
```

## Stale Lead Cleanup

```text
Review these stale lead rules and draft a follow-up workflow. Keep external messages draft-first and flag any sensitive claims.
```

## Required Fields

```text
Create a required-field checklist for this sales pipeline. Include stage, owner, next action, last contact date, risk notes, and review owner.
```

## Task Routing

```text
Design CRM task routing for this workflow. Include trigger, owner logic, task fields, due date rules, review gates, and failure modes.
```

## Review Gate

```text
Identify where this CRM workflow needs human review before customer-facing messages, stage changes, or sensitive decisions.
```
