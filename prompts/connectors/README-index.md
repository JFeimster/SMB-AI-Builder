# Connector Prompt Index Note

Connector prompts should help the user plan before connecting tools.

## Recommended Prompt Order

1. Workflow audit
2. Connector stack map
3. Risk classification
4. Review gate design
5. Pilot plan
6. QA checklist

## Good Prompt Behavior

- Use sample data where possible.
- Keep sensitive workflows draft-first.
- Ask for missing context instead of inventing it.
- Identify what should stay human.

## Bad Prompt Behavior

- Asking users to paste private data unnecessarily
- Pretending an integration is already live
- Making legal, tax, HR, lending, underwriting, accounting, medical, or compliance decisions
- Claiming guaranteed outcomes
